# gui.new agent skill

Turn HTML into shareable URLs. One POST call, instant rendered page with real-time sync.

## Install

```bash
npx skills add stratuslabs/gui-new-skill --skill gui-new
```

Global install (available in all projects):

```bash
npx skills add stratuslabs/gui-new-skill --skill gui-new -g
```

## What it does

Once installed, your agent will automatically use gui.new whenever it generates visual output — dashboards, charts, forms, tables, diagrams, landing pages, interactive tools. Instead of dumping HTML in chat, it POSTs to gui.new and shares a live URL.

## Works with

- Cursor
- Claude Code
- Codex
- OpenClaw
- Windsurf
- Any agent that supports the Skills protocol

## No account needed

The free tier requires no API key. Just POST HTML, get a URL.
